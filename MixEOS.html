EOS of mixture

Six different cubic equations of state are used to calculate molecular volumes of mixtures for both liquid and vapor phases. EOS's include Van der Waals, RK, SRK, PR, PRSV1 and PRSV2. 

- Requires downloaded 'Components' database -

Features:

  • Component data in database shared/accessible to all thermodynamic applications
  • Informs user if component is above critical temperature
  • Easily compare values from the six different equations of state
  • Information on the development of equations of state is presented
  • Instructions for the software application is presented
